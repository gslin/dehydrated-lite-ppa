# dehydrated-lite-ppa

PPA for dehydrated. (Original name is letsencrypt.sh)

## Installation

In Ubuntu system:

    sudo add-apt-repository ppa:gslin/dehydrated-lite
    sudo apt update
    sudo apt install -y dehydrated-lite
